Conoce nuestro proceso de trabajo

Así podrás saber paso a paso la creación de tu sitio web.

Reunión de acuerdos

En esta fase se llevara acabo una reunión en donde platicaremos de las necesidades del proyecto y una vez aceptado el proyecto se requerirá del pago de anticipo que sera del 50% para poder iniciar el trabajo.

Planeación

Una vez aceptado el proyecto se procederá a organizar la información obtenida y como se implementará dicha información en el sitio web.

Fase de diseño

Una vez aceptado el proyecto se procederá a organizar la información obtenida y como se implementará dicha información en el sitio web.

Ronda de revisiones

En esta fase una vez que se le haya presentado el diseño previo a como se verá su sitio web podrás analizar y realizar cambios a nivel estético.

Fase de desarrollo

Una vez aprobado el diseño se procederá a convertir ese diseño en código para ser publicado en la web, además se realizarán las pruebas de funcionamiento necesarias antes de su lanzamiento.

Lanzamiento y entrega

Al final tu sitio será instalado en el alojamiento web que se haya comprado y publicado con su nombre de dominio. Una vez que tu sitio haya sido entregado tendrás 7 días para realizar el pago restante del servicio que será del 50%.
